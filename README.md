
# Hospital Appointment System (C++)

A console-based Hospital Appointment Management System developed in **C++** using **Object-Oriented Programming (OOP)** concepts.  
This project demonstrates real-world use of inheritance, polymorphism, and encapsulation in a simple and easy-to-understand way.

---

## 📌 Features
- View available doctors
- Add new patients
- Book appointments with doctors
- Cancel appointments
- View appointment history
- Doctor availability management
- Different doctor types with different consultation fees

---

## 🧠 OOP Concepts Used
- Classes and Objects  
- Inheritance  
- Polymorphism (virtual functions & method overriding)  
- Encapsulation  
- Dynamic memory allocation

